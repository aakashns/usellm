---

`UseLLM.speak` method generates a spoken version of a given text, essentially a Text-to-Speech (TTS) function. This function is useful for applications that require vocalizing written text.


### Syntax

```Python
response = UseLLM.speak(
  text
)
```

### Parameters

The `speak` function accepts an `options` object with the following properties:

- `text` (required): The text that needs to be converted to speech. 

### Return Value

The `speak` function returns a URL to the generated audio.
